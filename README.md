# nes_rs

A faithful Rust-based emulation of the [Nintendo Entertainment System](https://en.wikipedia.org/wiki/Nintendo_Entertainment_System).

# Features

- [NES 6502](https://www.nesdev.org/wiki/CPU) implementation
